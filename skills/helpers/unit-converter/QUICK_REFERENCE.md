## Temperature Conversions

### Formulas

- °C to °F: `F = (C × 9/5) + 32`
- °F to °C: `C = (F - 32) × 5/9`
- °C to K: `K = C + 273.15`
- K to °C: `C = K - 273.15`
- °F to K: `K = (F - 32) × 5/9 + 273.15`

## Dimensionless Numbers (Reference)

These are dimensionless groups commonly used in engineering:

### Reynolds Number
```
Re = ρvD/μ
```
- ρ: density (kg/m³)
- v: velocity (m/s)
- D: characteristic length (m)
- μ: dynamic viscosity (Pa·s)

**Interpretation:**
- Re < 2,300: Laminar flow
- Re > 4,000: Turbulent flow
- 2,300 < Re < 4,000: Transition region

### Froude Number
```
Fr = v/√(gL)
```
- v: velocity (m/s)
- g: gravitational acceleration (9.81 m/s²)
- L: characteristic length (m)

### Mach Number
```
Ma = v/c
```
- v: velocity (m/s)
- c: speed of sound (≈ 343 m/s in air at 20°C)

### Prandtl Number
```
Pr = ν/α = μCp/k
```
- ν: kinematic viscosity (m²/s)
- α: thermal diffusivity (m²/s)
- μ: dynamic viscosity (Pa·s)
- Cp: specific heat (J/(kg·K))
- k: thermal conductivity (W/(m·K))

## Using the Converter

### Python Example
```python
from converter import convert_flow, convert_pressure, convert_temperature

# Convert flow rate
flow_gpm = convert_flow(100, 'L/min', 'gal/min')
print(f"Flow: {flow_gpm:.2f} gpm")

# Convert pressure
pressure_bar = convert_pressure(150, 'psi', 'bar')
print(f"Pressure: {pressure_bar:.2f} bar")

# Convert temperature
temp_f = convert_temperature(85, 'degC', 'degF')
print(f"Temperature: {temp_f:.1f} °F")
```

### Command Line Example (with pint)
```bash
python -c "import pint; u = pint.UnitRegistry(); print((100*u('L/min')).to('gal/min'))"
```

## Tips for Unit Conversions

1. **Always carry units through calculations** - Don't strip units until the final answer
2. **Check dimensional consistency** - Verify that equations make sense dimensionally
3. **Use consistent unit systems** - Mixing SI and Imperial units increases error risk
4. **Round appropriately** - Match precision to measurement accuracy
5. **Document conversions** - Note conversion factors used for traceability
6. **Validate results** - Sanity check converted values against expected ranges

## Common Conversion Mistakes to Avoid

1. **Confusing gauge and absolute pressure**
   - Remember: Absolute = Gauge + Atmospheric

2. **Temperature vs. temperature difference**
   - Use different conversions for absolute temp vs. ΔT

3. **Confusing dynamic and kinematic viscosity**
   - Need density to convert between them

4. **Mixing mass and volumetric flow**
   - Use density to convert: ṁ = ρ × Q̇

5. **Forgetting squared or cubed units**
   - Area: multiply factor twice
   - Volume: multiply factor three times
